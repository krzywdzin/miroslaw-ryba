# Miroslaw Ryba

A modern React frontend for [MiroFish](https://github.com/666ghj/MiroFish) -- an LLM-powered social simulation platform for prediction and analysis. Replaces the original Chinese Vue interface with a clean Polish/English React SPA.

---

> ## **VIBE CODE ALERT**
>
> **This entire project was 100% vibe coded using [Claude Code](https://claude.ai/code) (Claude Opus 4.6).** No human wrote a single line of production code. Every component, hook, test, translation, and config file was generated by AI agents.
>
> **What does this mean for you?**
>
> - There **will** be bugs, edge cases, and rough spots that no human has ever reviewed
> - Code patterns may be inconsistent or over-engineered in places
> - Some features may not work correctly with the real MiroFish backend
> - Tests exist but may not cover real-world scenarios
> - Polish translations were generated, not reviewed by a native speaker
>
> **This is an experiment, not production software.** Treat it as a starting point, not a finished product.
>
> **We need your help!** If you find bugs, bad patterns, or things that don't make sense -- please open an issue or submit a PR. Every contribution makes this project better. Check [CONTRIBUTING.md](CONTRIBUTING.md) to get started.

## Features

- 5-stage prediction pipeline (Graph -> Environment -> Simulation -> Reports -> Chat)
- Interactive knowledge graph visualization with entity filtering
- Real-time dual-platform simulation monitoring (Twitter + Reddit)
- PDF and Markdown report export
- AI agent conversations (ReportAgent and simulated agents)
- Dark mode with OS preference detection
- Keyboard shortcuts for common actions
- Bilingual interface (Polish / English)
- Full configuration panel (API keys, models, Docker management)

## Quick Start

```bash
git clone https://github.com/OWNER/miroslaw-ryba.git
cd miroslaw-ryba
cp .env.example .env        # Edit with your API keys
docker compose up -d         # Start MiroFish backend
npm install && npm run dev   # Start frontend at http://localhost:5173
```
