## Typography

Inherited from Phase 02 UI-SPEC. No changes to base scale. Exactly 4 roles, 2 weights.

| Role | Size | Weight | Line Height | Phase 6 Usage |
|------|------|--------|-------------|---------------|
| Body | 15px | 400 (regular) | 1.5 | Post content text (truncated and expanded), expanded post body |
| Label | 13px | 600 (semibold) | 1.4 | Agent names, action type badges, round labels, timeline event text, counter labels, metadata (timestamps, elapsed time -- rendered in weight 400 and muted color), "wiecej..." expand link |
| Heading | 22px | 600 (semibold) | 1.2 | Counter values (post count, agent count) in progress bar |
| Display | 30px | 600 (semibold) | 1.1 | Not used |

Usage notes:
- Metadata text (timestamps, elapsed time) uses Label size (13px) at weight 400 with muted color to visually differentiate from semibold labels.
- Counter values in the progress bar use Heading size (22px) to create visual prominence.
- Truncated post content uses Body size (15px) with `line-clamp-2`, same size as expanded content.

## Component Specifications

### SimulationProgressBar (sticky top bar)

- Height: 56px, white background, bottom border 1px
- Sticky: `position: sticky; top: 0; z-index: 10`
- Left section: "Runda X/Y" label (13px semibold) + Progress component (accent fill, 120px wide)
- Center section: activity counters in a row with 24px gaps
  - Each counter: value (22px semibold / Heading role) over label (13px regular muted / Label role at weight 400)
  - Counters: posty (posts), komentarze (comments), debaty (debates), agenci (active agents)
- Right section: elapsed time (13px muted / Label role at weight 400) + action button
  - Running state: "Zatrzymaj" (Stop) button, destructive variant, outline style
  - Completed state: "Zakonczone" success badge + "Przejdz do raportu" accent button
  - Failed state: "Blad symulacji" error badge + "Wroc do srodowiska" outline button

### PostCard (shared between Twitter and Reddit)

- Border-bottom only (no full card border) for feed density
- Padding: 16px horizontal (md), 8px vertical (sm)
- Row 1 (metadata): platform icon (16px, platform color) + agent name (13px semibold / Label) + round badge (Badge variant="outline", "R3") + action type badge (Badge variant="secondary") + timestamp (13px regular muted / Label at weight 400, right-aligned)
- Row 2 (content): post text (15px regular / Body), line-clamp-2 when collapsed
- Row 3 (expand): "wiecej..." link (13px / Label, accent color) when content exceeds 2 lines
- Highlight state: bg-accent/10, ring-1 ring-accent/50 (when timeline event clicked)
- Entrance animation: slide-in-from-top-2 + fade-in, 200ms ease-out (for newly arrived posts)
- Memoized with React.memo to prevent unnecessary re-renders

### RedditThread (nested comments)

- Indent: 16px (md) per nesting level via marginLeft
- Max visual depth: 4 levels (64px max indent)
- Thread connector: 1px solid border-left on indent column, muted color
- Depth > 4: flatten visually, show "w odpowiedzi na @agent" (replying to) text instead

### NewPostsIndicator (floating banner)

- Appears when user scrolled down and new posts arrive
- Position: sticky top of feed column, below column header
- Background: accent color, white text, 13px semibold (Label role)
- Copy: "X nowych postow" (with Polish plural forms)
- Click action: smooth scroll to top
- Z-index: 5 (above feed, below progress bar)

### EventTimeline (right column)

- Grouped by round, newest round at top
- Round header: "Runda N" (13px semibold / Label), sticky within scroll
- Event items below each round header:
  - Debate: Swords icon (amber-500) + agent names
  - Stance change: ArrowUpDown icon (accent) + agent name + direction
  - Post batch: FileText icon (muted) + "N postow" count
- Click on round: sets activeRoundFilter in store, filters both feeds
- Click on event: sets highlightedEventId, highlights related posts in feeds
- Active round: accent background at 10% opacity, accent left border 3px
- Clear filter: "Wszystkie rundy" (All rounds) button at top, visible when filter active

### CompletionBar (replaces progress bar after simulation ends)

- Same dimensions as SimulationProgressBar (56px, sticky)
- Background: success green at 5% opacity
- Left: "Zakonczone" badge (success variant) + final round count
- Center: final counters (same layout as running state, but static)
- Right: "Przejdz do raportu" button (accent, primary variant)

## Interaction Contract

### Feed Behavior

| Behavior | Specification |
|----------|---------------|
| New post arrival | Prepend at top of feed. If user at scroll top: auto-scroll to show. If scrolled down: show NewPostsIndicator. |
| Post expand | Click "wiecej..." toggles line-clamp off. No animation (instant). |
| Feed scroll | Independent per column via ScrollArea. No cross-column scroll sync. |
| Post after completion | Feed remains fully browsable. No auto-redirect. |

### Timeline Interactions

| Behavior | Specification |
|----------|---------------|
| Click round | Set activeRoundFilter. Both feeds filter to show only that round's posts. Timeline round gets accent highlight. |
| Click event | Set highlightedEventId. Related posts in feeds get highlight style (bg-accent/10, ring). Feed does NOT auto-scroll to highlighted posts. |
| Clear filter | Click "Wszystkie rundy" button or click active round again (toggle off). |

### Scroll Anchor Detection

| Condition | Result |
|-----------|--------|
| scrollTop < 50px | User is "at top" -- new posts auto-visible |
| scrollTop >= 50px | User is "scrolled down" -- show NewPostsIndicator |
| Click NewPostsIndicator | Smooth scroll to top (behavior: 'smooth') |

### Route Guard

| Condition | Action |
|-----------|--------|
| No simulationId in environment store | Redirect to /environment |
| Simulation not started/not found | Show empty state with spinner + "Laczenie z symulacja..." |

## Animation Contract

Inherits Phase 02 base animations. Phase 6 additions:

| Element | Animation | Duration | Easing |
|---------|-----------|----------|--------|
| New post entrance | slide-in-from-top-2 + fade-in | 200ms | ease-out |
| Post highlight (timeline click) | background-color + box-shadow transition | 150ms | ease-out |
| Progress bar fill | width transition | 300ms | ease-out |
| Counter value change | No animation (instant update) | -- | -- |
| NewPostsIndicator appear | fade-in + slide-in-from-top-1 | 150ms | ease-out |
| Timeline Sheet (responsive) | slide-in-from-right (shadcn Sheet default) | 200ms | ease-out |

Respects `prefers-reduced-motion: reduce` -- disable entrance animations, keep opacity fades only.

## Loading States

| Context | Pattern | Details |
|---------|---------|---------|
| Initial page load | Centered spinner | Spinner (16px) + "Laczenie z symulacja..." text while first poll resolves |
| Progress bar data loading | Skeleton | 56px skeleton bar matching progress bar dimensions |
| Feed columns empty (no posts yet) | Empty column | Muted text: "Oczekiwanie na posty..." / "Waiting for posts..." centered in column |
| Timeline empty | Empty column | Muted text: "Oczekiwanie na zdarzenia..." / "Waiting for events..." |
| Poll failure (single) | Silent retry | TanStack Query auto-retries 3x. No visual indicator for single failure. |
| Poll failure (exhausted) | Toast | Error toast with "Utracono polaczenie z symulacja" + retry button |
