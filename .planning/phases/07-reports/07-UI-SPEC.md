## Spacing Scale

Inherited from Phase 2. No changes.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, TOC item padding |
| md | 16px | Default element spacing, toolbar button gaps |
| lg | 24px | Section padding, sidebar TOC padding |
| xl | 32px | Content area padding, report article margins |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Report sidebar (TOC) width: 240px, sticky positioned
- Report article max-width: 720px (prose reading width)
- Report toolbar height: 48px
- Section progress list item height: 36px
- History table row height: 48px

## Typography

Inherited from Phase 2. Weights 400 and 600 only.

| Role | Size | Weight | Line Height | Usage in Phase 7 |
|------|------|--------|-------------|-------------------|
| Body | 15px | 400 (regular) | 1.5 | History table cells, progress section names, report body fallback |
| Label | 13px | 600 (semibold) | 1.4 | Toolbar button labels, TOC section links, table headers, status badges |
| Heading | 22px | 600 (semibold) | 1.2 | Page titles ("Raport", "Historia symulacji"), generation heading |
| Display | 30px | 600 (semibold) | 1.1 | Not used in Phase 7 |

### Prose Typography (react-markdown content)

Prose styling is handled by `@tailwindcss/typography` via the `prose` utility class. Use `prose-slate` variant for neutral color scheme.

| Element | Effective Size | Effective Line Height |
|---------|---------------|----------------------|
| prose p | 16px | 1.75 |
| prose h1 | 36px | 1.1 |
| prose h2 | 24px | 1.3 |
| prose h3 | 20px | 1.6 |
| prose li | 16px | 1.75 |
| prose table td | 14px | 1.5 |
| prose code | 14px | 1.5 |

Apply `max-w-none` to the prose container to allow full-width tables within the 720px article column.

## Color

Inherited from Phase 2. Light mode only (dark mode deferred to Phase 9).

| Role | Value | Usage in Phase 7 |
|------|-------|-------------------|
| Dominant (60%) | `hsl(0 0% 100%)` / white | Page background, report article background |

- "Otworz" link in history table
- Progress bar fill color
- In-progress section spinner color

### Semantic Colors (Phase 7 Usage)

| State | Value | Usage |
|-------|-------|-------|
| Success | `hsl(142 71% 45%)` / green-500 | Completed section checkmark, "Ukonczony" badge in history |

| Error | `hsl(0 84% 60%)` / red-500 | "Blad" badge in history, generation failure alert |

## Layout Contract

### Report View Page (`/report/:simulationId`)

```
+--[Header 56px]---------------------------------------------+
| [Logo]  [Breadcrumb: Raport > ...]      [Lang] [Config] [*] |
+----------+--------------------------------------------------+
| Sidebar  | Toolbar (48px)                                    |
| 280px    | [Eksportuj PDF] [Pobierz Markdown]               |
|          +--------------------------------------------------+
|          | Report Article        | TOC Sidebar              |
| [Steps]  | max-w: 720px          | width: 240px             |
|          | padding: 32px         | sticky top: 104px        |
|          | prose prose-slate      | bg: secondary            |
|          |                       | padding: 16px            |
|          |                       | [* Active section]       |
|          |                       | [  Section 2]            |
|          |                       | [  Section 3]            |
+----------+--------------------------------------------------+
| [Footer 40px]                                               |
+-------------------------------------------------------------+
```

- Toolbar sits between header and content, full width of content area, bottom border 1px
- Report article and TOC sidebar are side by side within the content area
- TOC sidebar is sticky (top offset = header 56px + toolbar 48px = 104px)
- TOC sidebar has `max-height: calc(100vh - 104px - 40px)` with overflow-y auto
- Content scrolls naturally; TOC stays fixed in viewport

### Report Generation Page (same route, generating state)

```
+--[Header 56px]---------------------------------------------+
| [Logo]  [Breadcrumb: Raport > Generowanie]  [Lang] [Config] |
+----------+--------------------------------------------------+
| Sidebar  | Centered content (max-w: 480px)                  |
| 280px    |                                                   |
|          |   [Heading: Generowanie raportu]                  |
|          |   [Progress bar: 45%]                             |
|          |                                                   |
|          |   [v] Analiza wynikow                             |
|          |   [*] Generowanie prognoz...                      |
|          |   [ ] Podsumowanie                                |
|          |   [ ] Rekomendacje                                |
|          |                                                   |
+----------+--------------------------------------------------+
| [Footer 40px]                                               |
+-------------------------------------------------------------+
```

- Vertically centered within content area
- Progress bar spans full width of container (480px)
- Section list uses 36px row height with icon (16px) + label

### History Page (`/history`)

```
+--[Header 56px]---------------------------------------------+
| [Logo]  [Breadcrumb: Historia symulacji]    [Lang] [Config] |
+----------+--------------------------------------------------+
| Sidebar  | [Heading: Historia symulacji]                     |
| 280px    |                                                   |
|          | +--Table-----------------------------------------+ |
|          | | Temat     | Status      | Data   | Agenci | R | |
|          | |-----------|-------------|--------|--------|---| |
|          | | Topic 1   | [Ukonczony] | 22 mar | 5      | O | |
|          | | Topic 2   | [Blad]      | 21 mar | 3      | - | |
|          | | Topic 3   | [W trakcie] | 20 mar | 8      | - | |
|          | +----------------------------------------------+ |
|          |                                                   |
|          | [Empty state if no simulations]                   |
+----------+--------------------------------------------------+
| [Footer 40px]                                               |

```

- Table uses full content width
- Sortable by date column (click header to toggle asc/desc)
- Status column uses Badge component with color map from Color section
- Report column: "Otworz" as text link (accent color) if report exists, "Brak raportu" in muted text if not
- Row height: 48px
- Table header: Label typography (13px semibold)

### Responsive Behavior (1024px - 1279px)

- App sidebar collapses to 64px icon rail (existing behavior)
- Report TOC sidebar collapses below article content as a horizontal section list
- Report article takes full content width (minus padding)
- History table remains full width, horizontal scroll if needed

## Loading States

| Context | Pattern | Details |
|---------|---------|---------|
| Report content loading | Skeleton | 3 paragraph-shaped skeleton blocks (full width, varying heights: 120px, 80px, 160px) + sidebar TOC skeleton (5 lines at 16px height) |

## Error States

| Context | Pattern | Details |

## Animation Contract

Inherited from Phase 2 defaults. Phase 7 additions:

| Element | Animation | Duration | Easing |
|---------|-----------|----------|--------|
| TOC active section transition | Background color + left border | 150ms | ease-out |
| Smooth scroll to section | Native smooth scroll | browser default | `scroll-behavior: smooth` |
| Progress bar value change | Width transition | 300ms | ease-out |
| Section checkmark appear | Scale from 0 to 1 + fade in | 200ms | ease-out |
| Generation complete transition | Fade out progress, fade in report | 300ms | ease-in-out |

Respect `prefers-reduced-motion: reduce` -- disable all except opacity fades.

## Print CSS Contract (PDF Export)

```css
@media print {
  /* Hide app shell */
  header, nav, aside, footer, .no-print { display: none !important; }

  /* Hide report TOC sidebar */
  .report-toc { display: none !important; }

  /* Hide toolbar */
  .report-toolbar { display: none !important; }

  /* Full width for report content */
  main { padding: 0 !important; margin: 0 !important; }
  .report-article { max-width: 100% !important; margin: 0 !important; }

  /* Page break control */
  h1, h2, h3 { break-after: avoid; }
  table, figure, pre { break-inside: avoid; }

  /* Clean background */
  body { background: white !important; color: black !important; }

  /* Remove link decorations */
  a { color: inherit !important; text-decoration: none !important; }
}
```

Add `no-print` class to: app sidebar, header, footer, report toolbar, TOC sidebar.

## Scroll Spy Contract

| Property | Value |
|----------|-------|
| Observer root | `null` (viewport) |
| Root margin | `-80px 0px -60% 0px` (accounts for header + toolbar, biases toward top of viewport) |
| Threshold | `0` (triggers as soon as element enters/leaves observation area) |
| Observed elements | All `h2` and `h3` elements within report article |
| Active state | First intersecting heading from top wins |
| Initial state | First heading in outline list |
| Re-initialization | When heading list changes (after markdown renders) |

TOC active section styling:
- Background: `hsl(217 91% 60% / 0.08)`
- Left border: 3px solid accent
- Text: accent color, semibold weight
- Padding-left: 12px (indented from border)
- Transition: background-color 150ms ease-out
