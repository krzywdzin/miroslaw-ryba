       ```html
       <script>
         (function() {
           var t = localStorage.getItem('theme');
           var dark = t === 'dark' || (!t && window.matchMedia('(prefers-color-scheme: dark)').matches);
           if (dark) document.documentElement.classList.add('dark');
         })();
       </script>
       ```

       - applyTheme() toggles 'dark' class on document.documentElement
       - useTheme() returns { theme, setTheme, resolvedTheme }
       - useState initializer reads from localStorage, defaults to 'system'
       - setTheme callback: updates state, writes/removes localStorage, calls applyTheme
       - useEffect: applies theme on mount, listens to matchMedia 'change' event for system theme updates
