    **4. Create `src/features/graph/components/UploadForm.tsx`:**
    - Combines UploadDropzone + FileList + PredictionInput + submit button
    - Local state for `files: File[]` (NOT in store -- File objects not serializable)
    - predictionGoal from useGraphStore
    - Submit button text: `t('graph:upload.submit')` = "Buduj graf"
    - Disabled when: no files OR empty prediction goal OR isSubmitting
    - On submit:
      1. Set isSubmitting=true
      2. Build FormData: `formData.append('file', file)` for each file + `formData.append('simulation_requirement', predictionGoal)`
      3. Call `graphApi.generateOntology(formData)` -> get `project_id`
      4. Store projectId via `useGraphStore.getState().setProjectId(project_id)`
      5. Call `graphApi.buildGraph(project_id)` -> get `task_id`
      6. Store taskId, set step to 'building'
      7. Navigate to `/graph/view` using `useNavigate()`
    - On error: show ErrorAlert with retry, use `t('graph:build.error')`
    - Use LoadingSpinner in button during submission
