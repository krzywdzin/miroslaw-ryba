# Phase 8 — UI Design Contract

> Visual and interaction contract for the interactive dialogue chat interface. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn (initialized, default style) |
| Preset | default (neutral baseColor, CSS variables enabled, Tailwind v4) |
| Component library | Radix UI (via shadcn/ui) |
| Icon library | Lucide React |
| Font | Geist Sans (`@fontsource-variable/geist`), weights 400 + 600 |

### shadcn Components Required for Phase 8

Already installed (reuse from prior phases):
- `select` -- Agent dropdown with groups and separator
- `scroll-area` -- Chat message scroll container
- `collapsible` -- Tool calls expandable section
- `badge` -- Platform badges (Twitter/Reddit)
- `button` -- Send button, retry button
- `separator` -- Agent group divider in dropdown
- `skeleton` -- Loading state for profile panel
- `alert` -- Error display fallback
- `sheet` -- Existing pattern, not directly needed

No new shadcn components required.

## Spacing Scale

Inherited from Phase 2. All values are multiples of 4.

| Token | Value | Usage in Phase 8 |
|-------|-------|-------------------|
| xs | 4px | Gap between typing indicator dots, icon-to-text inline gap |
| sm | 8px | Chat bubble internal icon gap, message timestamp offset |
| md | 16px | Chat bubble padding (px-4 py-3), message list gap between bubbles |
| lg | 24px | Agent context panel section padding, chat area padding |
| xl | 32px | Split panel gap between chat and context panel |
| 2xl | 48px | Not used in Phase 8 |
| 3xl | 64px | Not used in Phase 8 |

Exceptions:
- Chat bubble max-width: 80% of chat column width
- Agent context panel width: 320px fixed on desktop
- Chat input area height: 56px minimum (grows with multiline content, max 120px)
- Typing indicator bubble width: max 80px

## Typography

Inherited from Phase 2. Geist Sans, weights 400 and 600 only.

4 sizes: 13, 15, 22, 30.

| Role | Size | Weight | Line Height | Usage in Phase 8 |
|------|------|--------|-------------|-------------------|
| Body | 15px | 400 (regular) | 1.5 | Chat message text (user and agent), agent description, personality text |
| Label | 13px | 600 (semibold) | 1.4 | Agent selector label, "Na podstawie:" header, profile section headers, timestamps |
| Heading | 22px | 600 (semibold) | 1.2 | Not used in Phase 8 (page title is in stepper) |

| Element | Size | Weight | Notes |
|---------|------|--------|-------|

| Agent message prose | 15px (prose) | 400 | Rendered via react-markdown + remark-gfm, uses prose sizing (not prose-sm) |

## Color

Light mode only. Dark mode deferred to Phase 9. Inherited from Phase 2.

| Role | Value | Usage in Phase 8 |
|------|-------|-------------------|
| Dominant (60%) | `hsl(0 0% 100%)` / white | Chat area background, input area background |
| Secondary (30%) | `hsl(240 5% 96%)` | Agent context panel background, agent message bubbles (bg-muted) |
| Accent (10%) | `hsl(217 91% 60%)` / blue-500 | See reserved list below |
| Destructive | `hsl(0 84% 60%)` / red-500 | Error message bubbles only |

### Accent Reserved For (Phase 8)

- User message bubbles (bg-primary, text-primary-foreground -- white text on accent background)
- Send button (accent background, white icon)
- Active agent selection indicator in dropdown

### Semantic Colors

| State | Value | Usage in Phase 8 |
|-------|-------|-------------------|
| Muted text | `hsl(240 4% 46%)` | Timestamps, empty state body, placeholder text in input |
| Muted background | `hsl(240 5% 96%)` | Agent message bubbles, typing indicator bubble, context panel bg |
| Error inline | `hsl(0 84% 60%)` | Failed message indicator, error toast |

### Chat Bubble Colors

| Bubble Type | Background | Text Color | Border |
|-------------|------------|------------|--------|
| User message | `bg-primary` (accent) | `text-primary-foreground` (white) | none |
| Agent message | `bg-muted` (secondary) | `text-foreground` (default) | none |
| Error message | `bg-destructive/10` | `text-destructive` | 1px border-destructive/20 |
| Typing indicator | `bg-muted` (secondary) | n/a (dots are muted-foreground/60) | none |

### Border

- Chat bubble border-radius: 16px (rounded-2xl) for natural bubble feel
- Input field border-radius: 8px (rounded-lg), 1px border default
- Agent context panel: left border 1px separator, no rounded corners (flush with layout edge)
- Source/tool call section border-radius: 8px, 1px muted border

### Focal Point

The primary focal point is the **chat message area** -- the scrollable conversation log occupies the largest visual area and draws the eye through content flow and the accent-colored user bubbles contrasting against the neutral background.

## Component Inventory

### ChatBubble

| Variant | Alignment | Background | Content Rendering |
|---------|-----------|------------|-------------------|
| User | Right-aligned (ml-auto) | bg-primary | Plain text (15px) |
| Agent | Left-aligned (mr-auto) | bg-muted | Markdown via react-markdown + remark-gfm (prose, 15px) |
| Error | Left-aligned (mr-auto) | bg-destructive/10 | Error text with retry prompt |

Max-width: 80% of container. Border-radius: 16px (rounded-2xl). Padding: 16px horizontal, 12px vertical (px-4 py-3).

### AgentSelector

- shadcn Select component at top of chat column
- Label prefix: "Rozmowa z:" (13px semibold)
- Two groups separated by SelectSeparator:
  - Group 1 ("Asystent"): ReportAgent (default)
  - Group 2 ("Agenci symulacji"): List of simulated agent names from profiles
- If no profiles loaded: Group 2 shows disabled placeholder "Brak agentow" / "No agents"

### ChatInput

- Textarea (not single-line input) for multiline support
- Placeholder: "Napisz wiadomosc..." / "Write a message..."
- Send button: icon-only (Send icon from Lucide), 36px square, accent background, rounded-md
- Send button disabled when: input empty OR mutation pending
- Send button `aria-label`: "Wyslij wiadomosc" / "Send message"
- Textarea grows from 1 line to max 4 lines (56px to 120px), then scrolls internally
- Submit on Enter (without Shift). Shift+Enter inserts newline.

### TypingIndicator

- Three bouncing dots (8px diameter circles) in a muted bubble
- CSS-only animation: `animate-bounce` with staggered delays (0ms, 150ms, 300ms)
- Bubble styled identically to agent message bubble but max-width 80px
- Appears left-aligned when mutation is pending, disappears when response arrives

### AgentContextPanel

**For simulated agents:**

| Section | Content | Typography |
|---------|---------|------------|
| Name | Agent name from profile | 15px semibold |
| Platform | Badge with Twitter bird or Reddit icon | Badge component, 13px |
| Personality | PersonalitySchema text, full display | 15px regular, muted-foreground for label |
| Stance | StanceSchema text | 15px regular |
| Memory | Placeholder text if unavailable; display if profile includes memory data | 15px regular |

**For ReportAgent:**

| Section | Content | Typography |
|---------|---------|------------|
| Name | "ReportAgent" | 15px semibold |
| Icon | Bot icon (Lucide) at 48px, accent color | Centered above name |
| Description | "Asystent raportu predykcji. Odpowiada na pytania o wyniki symulacji i analize." | 15px regular |

No personality, stance, platform, or memory sections for ReportAgent.

### SourcesList

- Appears below the agent message bubble content, inside the same bubble
- Header: "Na podstawie:" in 13px semibold
- Source items: bulleted list, 13px regular
- Separated from message content by a thin border-top (1px muted)
- Only rendered when `sources` array is non-empty

### ToolCallsSection

- Appears below SourcesList (or below message content if no sources)
- Uses shadcn Collapsible component
- Trigger: "Wywolania narzedzi" label with ChevronDown icon (rotates on open)
- Content: monospace text (font-mono, 13px) showing tool call details
- Collapsed by default
- Only rendered when `toolCalls` array is non-empty

## Interaction States

### Chat Message Flow

| State | Visual Treatment |
|-------|-----------------|
| Idle (no messages) | Empty state: centered heading + body text + MessageCircle icon (48px, muted) |
| User typing | Textarea shows input, send button becomes active (accent color) |
| Message sent | User bubble appears immediately, typing indicator shows in agent position |
| Response received | Typing indicator replaced by agent bubble with markdown content |
| Send failed | Error bubble appears with inline retry, toast notification |
| Agent switching | All messages cleared instantly, empty state shown for new agent |

### Loading States

| Context | Pattern | Details |
|---------|---------|---------|
| Profile list loading | Skeleton | 3 skeleton items in the simulated agents group of Select |
| Agent context panel loading | Skeleton | Skeleton blocks matching panel section heights |
| Waiting for response | Typing indicator | Animated three-dot bounce in muted bubble |
| Send button during request | Disabled + spinner | 16px spinner replaces Send icon, button disabled |

### Error States

| Context | Pattern | Details |
|---------|---------|---------|
| Chat send failure | Inline error bubble + toast | Error bubble in chat flow, sonner toast for notification |
| Profile fetch failure | ErrorAlert in context panel | Reuse shared ErrorAlert with retry |
| No simulation selected | Route guard redirect | Redirect to /simulation or /graph if no simulation_id |

## Animation Contract

Inherited from Phase 2 base, with Phase 8 additions:

| Element | Animation | Duration | Easing |
|---------|-----------|----------|--------|
| New message appear | Fade in + slide up 8px | 150ms | ease-out |
| Typing indicator dots | Bounce (staggered) | 600ms per cycle | ease-in-out, infinite |
| Typing indicator appear/disappear | Fade in/out | 100ms | ease-out |
| Agent context panel content swap | Fade in | 150ms | ease-out |
| Collapsible tool calls expand | Height reveal | 200ms | ease-out (shadcn default) |
| Send button icon swap (send/spinner) | Opacity crossfade | 100ms | linear |

Respect `prefers-reduced-motion: reduce` -- disable bounce animation, keep only opacity fades.
